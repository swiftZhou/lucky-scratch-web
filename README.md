## Enable GitHub Pages (once)

1. https://github.com/swiftZhou/lucky-scratch-web/settings/pages
2. Source: **Deploy from a branch**
3. Branch: **`main`**, Folder: **`/docs`**
4. **Save** — wait 2–5 minutes

**Do not use GitHub Actions** as the Pages source.

## Update site

```bash
# edit files under docs/
git add docs/
git commit -m "Update site"
git push origin main
```
